#　このリポジトリについて
このリポジトリは highmetal が趣味で制作したアプリやサイトをまとめているものです。


#### 公開URL
![logo](image/high.png) **https://highmetal.github.io**

---

### サイトとアプリ一部紹介
* [自己紹介](https://highmetal.github.io/src/aboutMe) 
highmetalの自己紹介サイト。

* [チェックリスト](https://highmetal.github.io/src/checkList)
買い物のメモやToDoListに使える便利なリスト。

* [カード金額計算（cac）](https://highmetal.github.io/old/cardAmountCalculation)
トレーディングカードを購入する時にメモをしたり合計金額を確かめられるサイト。近日アップデート予定。

* [HTMLPreview](
https://highmetal.github.io/src/htmlPreview)
ブラウザでHTMLを試すことができるサイト。スマートフォンでも簡単に入力できる補完機能があり、外出中でもコーディングできる。

* [ルーレット](https://highmetal.github.io/src/roulette)
シンプルなルーレットアプリ。
